# MERN Contract Farming Platform

A comprehensive solution for modern contract farming contracts, tracking, and marketplace.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Lucide Icons, Framer Motion.
- **Backend**: Node.js, Express, MongoDB, JWT Auth.

## Getting Started

### Backend
1. `cd server`
2. `npm install`
3. `npm run dev`

### Frontend
1. `cd client`
2. `npm install`
3. `npm run dev`

## Features
- Secure Authentication
- Role-based Dashboards
- Contract Lifecycle Management
- Mock AI & Weather Services
